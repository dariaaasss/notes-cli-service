# Notes CLI

Консольное приложение для управления заметками с поддержкой интерактивного и CLI режимов.

## Возможности

- Создание заметок с заголовком и содержанием
- Просмотр всех заметок в виде таблицы
- Поиск заметок по тексту
- Редактирование существующих заметок
- Удаление заметок
- Интерактивный режим с меню
- CLI режим с флагами для скриптов и автоматизации

## Установка

```bash
# Клонирование репозитория
git clone https://github.com/dariaaasss/notes-cli-service.git
cd notes-cli

# Создание виртуального окружения
python -m venv .venv

# Активация окружения
# Windows:
.venv\Scripts\activate
# Linux/macOS:
source .venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt
```

## Использование

### Интерактивный режим

Запустите без аргументов для интерактивного меню:

```bash
python -m src.main
```

В интерактивном режиме доступны опции:
- Показать все заметки
- Поиск по заметкам
- Добавить новую заметку
- Редактировать заметку
- Удалить заметку

### CLI режим

#### Добавление заметки

```bash
python -m src.main add --title "Заголовок" --msg "Текст заметки"
```

#### Просмотр всех заметок

```bash
python -m src.main list
```

#### Поиск заметок

```bash
python -m src.main list --query "поисковый запрос"
# или
python -m src.main list -q "запрос"
```

#### Редактирование заметки

```bash
python -m src.main edit --id "id-заметки" --title "Новый заголовок" --msg "Новый текст"
```

#### Удаление заметки

```bash
python -m src.main delete --id "id-заметки"
```
